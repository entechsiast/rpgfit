# /plan

Creates a set of GitHub Issues for a given goal and adds them to the project board.

## Usage

```
/plan <goal description>
```

## Process

1. Analyze the goal and break it into discrete, actionable tasks
2. For each task, create a GitHub Issue in `entechsiast/rpgfit` with:
   - Clear title and description
   - Acceptance criteria
   - Appropriate labels
3. Add each issue to project #2
4. Set Status, Agent, Effort fields
5. Report back with links to all created issues
